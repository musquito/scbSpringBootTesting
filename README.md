## How to run the application?
~~~
git clone https://github.com/peasunseries/book-store.git
cd book-store
mvn clean install
mvn spring-boot:run

~~~


** If mvn clean install , The Unit Test Failed pleas ignore test

~~~
 mvn clean install -DskipTests
~~~
